# Editorial Intelligence —— 编辑情报

> 一份为高管而设计的视觉语言。
> 不是装饰文档，是把"数据本身"做成可被信任的视觉证据。

## 哲学定位

**Editorial Intelligence** 是一种把"严肃新闻编辑部 + 战略咨询报告 + 金融终端"三者熔炼成的视觉范式。它不取悦读者，它**让读者无法回避数据本身**。它的灵感来自 *The Economist* 的图表语法、*Financial Times* 的数据新闻、McKinsey & BCG 的战略备忘录、Bloomberg Intelligence 的终端配色。它不追求"漂亮"，它追求"被相信"。每一处留白、每一个对齐、每一个数字的字号选择，都是经过无数次校对之后留下的——这是经验丰富的版面编辑、信息设计师、数据记者用毕生功力打磨出来的视觉秩序。

## 视觉语法

整本报告活在一套**强栅格秩序**里：12 列瑞士网格作为隐形脚手架，每个图表、每个数字、每个段落都对齐到这套尺度之上，呈现一种近乎建筑学的克制。版面里没有任何视觉噪音——没有渐变、没有阴影、没有装饰花纹、没有图标艺术。**数据本身就是图形**，标签是数据的脚注。图表区域占据 70% 的版面权重，文字降到次要位置；当文字出现时，它必须以"结论一句话 + 数据佐证"的形式存在，绝不允许出现解释性的大段文字。报告中所有大数字以**衬线 + 半粗体**呈现，传递一种"经过审计的庄严"；所有标签、注释、数据维度名以**无衬线细体**呈现，让数字本身成为唯一被强调的视觉主角。

## 色彩语言

整本报告锁定一套**克制而明确**的有限调色板：奶白底色（#F8F5EE，纸张感而非屏幕感）、炭灰主文（#1A1A1A）、深湖蓝作为品牌主色（#1B3A5C）、暖朱红作为风险信号（#C0392B）、琥珀黄作为警示中等优先级（#D89614）、墨绿作为正向健康指示（#2E6B47）。颜色不是装饰，颜色**就是分类系统本身**——读者扫视一页时，红、黄、绿三色立即告诉他"什么紧急、什么需要关注、什么在正常运转"。除此之外**禁止使用任何其他颜色**。这种限制是大师级编辑设计的核心纪律：每多一种颜色都是对信任的损耗。

## 节奏与呼吸

整份报告以**章节卡片 + 信息密度梯度**的方式组织节奏：每一页有且只有一个核心命题，命题以一句不超过 22 字的"裁判式结论"开篇，紧接着是用以证明结论的图表矩阵，最后用一行行动指令收束。每页之间用大幅留白和一条细横线分隔，让读者眼睛得以"换气"。每个图表都遵循 *Financial Times* 教科书式的图表纪律：标题居左、副标题描述方法、数据来源永远位于右下角、坐标轴单位明示、刻度数字优先于刻度线条。**这是数据新闻学的最高规范**，是无数次反复打磨之后才能呈现出的优雅。

## 字体与版式

字体系统极度克制：标题与大数字采用**衬线字体**（首选 Source Serif / Noto Serif），传达一种"已被验证"的权威；正文与图表标签采用**无衬线字体**（首选 Source Sans / Noto Sans），保证密集信息的可读性。**不允许使用任何装饰艺术字体、手写体、艺术符号**——这与项目方的明确指令一致，也与编辑情报哲学一致。状态分类一律不使用 emoji，而采用 CSS 几何符号——◆ 菱形（红线）、● 实心圆（严重 / 痛感）、○ 空心环（改进），配合红黄绿三色承载分类语义；这既贯彻"零装饰符号"的纪律，也确保打印输出与跨平台渲染稳定一致。所有标题左对齐，所有数字右对齐（便于纵向扫读），所有表格用极细的横分隔线而非边框包裹——这是杂志页排版的金标准。

## 工匠承诺

这份报告的每一个像素都必须经得起放大检验：每条线的粗细经过权衡，每个色值都从有限调色板中精确取出，每张图表的标注位置都是为了避免数据被遮挡而反复挪动过的结果。任何看到这份报告的人——CEO、研发总监、市场总监、客服总监、法务总监——都应该立刻感知到这是**一位资深信息设计师在为他们工作**，而不是一份模板填空。这种"被精心服务"的感觉，本身就是这套视觉哲学最重要的输出。

## 复用承诺

这套哲学的最大价值在于**可周期性复用**。下一期 VOC 报告、下一款产品（V50 Pro、Brave 8、AKASO 360 等）、下一个市场（德国、日本、英国），都可以沿用同一套栅格、同一套配色、同一套图表语法、同一套字体系统。读者只需要看一眼版面，就能识别出这是 VOC 系列报告——这种**视觉一致性**正是高管型读者建立长期信任的基础。
